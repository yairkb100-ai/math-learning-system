## מה זה הסקיל?

**סקיל "Course Writer"** הוא כלי אוטומטי שכותב לומדות מקצועיות בשביל בנות.

### מה הסקיל עושה?
```
INPUT:  "כתוב לי קורס על משוואות ריבועיות לתלמידי תיכון"
        ↓
PROCESS: הסקיל עובד לבד - כותב תוכן, מיצר תרגילים, מכין שאלות
        ↓
OUTPUT: קורס מלא עם:
        ✅ 5 פרקים מובנים
        ✅ תוכן מפורט בכל פרק
        ✅ 3-5 תרגילים בכל פרק
        ✅ 5 שאלות בחינה בכל פרק
        ✅ דוגמאות מתמטיות
        ✅ הסברים ברורים
```

### מה הסקיל **לא** עושה?
❌ לא מעלה את הקורס לאתר בשלך  
❌ לא מתרגם או מתאים קיים  
❌ לא עורך וידיאו

## שלב 1: תכנון הלומדה
### זמן: 5-10 דקות

**לפני שאתה קורא לסקיל**, עליך להחליט:**

### ❓ שאלה 1: מה הנושא המדויק?
```
דוגמאות:
✅ "משוואות ריבועיות"
✅ "אלגברה בסיסית לתלמידי כיתה ז'"
✅ "חשבון דיפרנציאלי"
✅ "מספרים מרוכבים"
```

**טיפ:** ככל שהנושא ספציפי יותר, הקורס טוב יותר.  
אל תגיד "מתמטיקה" — אמור "משוואות לינאריות ותכונותיהן".

### ❓ שאלה 2: לאיזו רמת קושי?
```
BEGINNER (למתחילים):
  - אין דרישות מקדימות
  - הכל מהתחלה
  - דוגמאות פשוטות
  → דוגמה: "משוואות ראשוניות לתלמידי כיתה ט'"

INTERMEDIATE (בינוני):
  - מושגים בסיסיים ידועים
  - התמקדות על יישום
  - דוגמאות מסובכות יותר
  → דוגמה: "משוואות ריבועיות בהקשר טריגונומטרי"

ADVANCED (מתקדם):
  - ידע עמוק בנושא
  - דיון בהוכחות
  - בעיות מחקר
  → דוגמה: "אלגברה לינארית — מרחבים וקטוריים"
```

**חשוב:** בחר את הרמה שמתאימה לתלמידיך!

### ❓ שאלה 3: כמה פרקים?
```
מינימום: 3 פרקים (קורס קצר)
מומלץ: 5 פרקים (קורס סטנדרטי)
מקסימום: 10 פרקים (קורס בנוגע)

טיפים לביחור:
- כל פרק = 45-60 דקות של לימוד
- 5 פרקים = קורס של 4-5 שעות
- 8 פרקים = קורס של 6-8 שעות

לתלמידי כיתה ט'-י: מומלץ 5-6 פרקים
```

### ❓ שאלה 4: מה להוסיף?
```
בחר 1-3 מהאפשרויות:

✅ Code Examples
   (למתמטיקה: נוסחאות Python, ודוגמאות חישוב)

✅ Diagrams
   (גרפים, דיאגרמות, ציורים של פונקציות)

✅ Case Studies
   (בעיות מהחיים האמיתיים שמשתמשות במתמטיקה)

✅ Resources
   (קישורים לאתרים, ספרות, כלים עזר)
```

### דוגמה: תכנון לומדה במתמטיקה

**נושא:** משוואות ריבועיות  
**רמה:** Intermediate (תלמידי כיתה י')  
**מספר פרקים:** 5  
**תוספות:** Diagrams + Code Examples  

✅ **קובץ תכנון:**
```
Course: "משוואות ריבועיות — מא"ב ל סופי סדרה"
Level: Intermediate
Chapters: 5
Include: Diagrams, Code Examples
Language: Hebrew
```

## שלב 4: עדכוני דקיקים
### זמן: 10-15 דקות (אם צריך)

**לאחר קבלת הקורס, אתה יכול לערוך:**

### 🔧 שינויים שאתה יכול לעשות:

#### 1️⃣ תיקון טעויות מתמטיות
```
אם מצאת טעות בנוסחה:

❌ לפני:
x² + 3x - 10 = 0
(שורשים: x = 2, x = -5)

✅ אחרי:
(תוקן ל-שורשים הנכונים)
```

#### 2️⃣ הוספת דוגמאות משלך
```
אם רוצה להוסיף דוגמה מממדרון או שיעור:

"בפרק 2, הוסף דוגמה על בעיות קנייה-מכירה
שנפתרות עם משוואות ריבועיות"
```

#### 3️⃣ שינוי קושי של תרגיל
```
אם תרגיל קשה מדי או קל מדי:

"שנה את תרגיל 3 בפרק 2 להיות יותר קל"
```

#### 4️⃣ הוספת עוד שאלות
```
"הוסף 3 שאלות בדיוק הבחינה בפרק 4"
```

### 💾 איך לשמור את הקורס?

**אפשרות 1: כ-JSON (למערכת המיוחדת שלך)**
```
הקבץ יישמר כ:
courses/quadratic-equations.json

ואח"כ ניתן להעלות למערכת דרך:
POST /api/courses/import
```

**אפשרות 2: כ-Markdown (קריאה קלה)**
```
# קורס: משוואות ריבועיות
## פרק 1: מבוא
...

שמור כ: קורס_משוואות_ריבועיות.md
```

**אפשרות 3: כ-PDF (הדפסה)**
```
המר JSON ל-PDF עם:
https://cloudconvert.com/json-to-pdf
```

## דוגמאות מעשיות

### דוגמה 1: קורס בסיסי על משוואות ריבועיות

**מה שאתה כותב:**
```
"כתוב קורס על משוואות ריבועיות לתלמידי כיתה י'.
ברמה בינונית, עם 5 פרקים.
כלול דיאגרמות של פונקציות וקוד Python."
```

**מה הסקיל יכתוב:**

```
📚 CHAPTER 1: מבוא למשוואות ריבועיות
├─ מה זו משוואה ריבועית?
├─ צורה כללית: ax² + bx + c = 0
├─ דוגמה: x² - 5x + 6 = 0
├─ שרטוט של הפרבולה (דיאגרמה)
├─ קוד Python:
│  import numpy as np
│  import matplotlib.pyplot as plt
│  a, b, c = 1, -5, 6
│  ...
├─ 4 תרגילים
└─ 5 שאלות בחינה

📚 CHAPTER 2: שיטת הנוסחה ודיסקרימיננט
📚 CHAPTER 3: פירוק לגורמים
📚 CHAPTER 4: יישומים בהנדסה ופיזיקה
📚 CHAPTER 5: בעיות תחרות וחידות
```

**זמן ביצוע: 25-30 דקות**
