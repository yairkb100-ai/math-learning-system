תרגול: מהו אחוז?

פרק 1 · אחוז = מאית: המרות בין שבר, עשרוני ואחוז

✏️ תרגילים

נסו לפתור לבד, ורק אחר כך גלו את הפתרון.

תרגיל 1: מאחוז לשבר קל
כתבו כל אחוז כשבר מצומצם:

א. 50%
ב. 25%
ג. 40%
ד. 8%
א. 50/100 = 1/2
ב. 25/100 = 1/4
ג. 40/100 = 2/5
ד. 8/100 = 2/25
תרגיל 2: מאחוז לעשרוני ובחזרה קל
המירו:

א. 35% לעשרוני
ב. 7% לעשרוני
ג. 0.62 לאחוז
ד. 0.04 לאחוז
א. 35\% = 0.35
ב. 7\% = 0.07 (זהירות — לא 0.7!)
ג. 0.62 = 62\%
ד. 0.04 = 4\%
תרגיל 3: משבר לאחוז בינוני
כתבו כל שבר כאחוז:

א. 12
ב. 34
ג. 710
ד. 920
ה. 1350
מרחיבים למכנה 100:

א. 50/100 = 50\%
ב. 75/100 = 75\%
ג. 70/100 = 70\%
ד. 45/100 = 45\%
ה. 26/100 = 26\%
תרגיל 4: האחוז המשלים בינוני
א. 45% מתלמידי בית הספר בנים. כמה אחוזים בנות?
ב. 82% מהמושבים באולם תפוסים. כמה אחוזים פנויים?
ג. שילמתי 30% מהחוב. כמה אחוזים נשארו לתשלום?
המשלים ל-100%:

א. 100\% - 45\% = 55\% בנות
ב. 100\% - 82\% = 18\% פנויים
ג. 100\% - 30\% = 70\% מהחוב נשארו
תרגיל 5: אתגר: סידור מעורב מאתגר
סדרו מהקטן לגדול: \ 42\%,\ 2/5,\ 0.44,\ 9/20,\ 39\%
נתרגם הכול לאחוזים:
2/5 = 40\%, \ 0.44 = 44\%, \ 9/20 = 45\%

מסדרים: 39\% < 40\% < 42\% < 44\% < 45\%

כלומר: 39\%,\ 2/5,\ 42\%,\ 0.44,\ 9/20
