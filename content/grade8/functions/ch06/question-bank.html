מאגר שאלות — פרק 6: עלייה וירידה

24 שאלות · קטעי עלייה, ירידה וקביעות בגרף של פונקציה · דף פתרונות בעמוד האחרון
שם: כיתה: תאריך:

חלק א' — מושגי יסוד 📖 (שאלות 1–4)

  1. באיזה כיוון קוראים גרף כדי לקבוע עלייה או ירידה?
  2. מה קורה לערכי ה-y בקטע שבו הפונקציה עולה?
  3. איך נראה הגרף בקטע שבו הפונקציה קבועה?
  4. פונקציה עולה כאשר x בין 0 ל-4 ויורדת בין 4 ל-9. באיזה x נמצאת הנקודה הגבוהה ביותר?

חלק ב' — עולה או יורדת? 🎢 (שאלות 5–8)

  1. כדור נזרק כלפי מעלה. מה עושה גרף הגובה שלו מיד לאחר הזריקה — עולה או יורד?
  2. בריכה מתרוקנת. מה עושה גרף כמות המים — עולה, יורד או קבוע?
  3. הטמפרטורה בלילה לא השתנתה כלל. איך נראה הגרף בשעות אלו?
  4. רץ מתרחק מביתו בריצה. מה עושה גרף המרחק שלו מהבית?

חלק ג' — נקודות שיא 🏔️ (שאלות 9–12)

  1. פונקציה עולה כאשר x בין 0 ל-5 ויורדת בין 5 ל-12. היכן הנקודה הגבוהה ביותר?
  2. פונקציה יורדת כאשר x בין 0 ל-3 ועולה בין 3 ל-10. היכן הנקודה הנמוכה ביותר?
  3. פונקציה עולה בין 0 ל-2, קבועה בין 2 ל-6 ויורדת בין 6 ל-9. באילו ערכי x מתקבל הערך הגבוה ביותר?
  4. פונקציה עולה בכל הקטע שבו היא מוגדרת. האם יש לה נקודת שיא באמצע הדרך?

חלק ד' — מסיפור לגרף 📝 (שאלות 13–16)

  1. בריכה מתמלאת ואז נשארת מלאה. תארו את צורת הגרף של כמות המים.
  2. נר דולק ומתקצר. האם גרף גובה הנר עולה או יורד?
  3. הולכים, עוצרים לנוח, וממשיכים ללכת באותו כיוון. תארו את גרף המרחק מנקודת המוצא.
  4. הטמפרטורה עולה בבוקר ויורדת בערב. תארו את צורת הגרף.

חלק ה' — השוואת ערכים ⚖️ (שאלות 17–20)

  1. פונקציה עולה כאשר x בין 1 ל-6. מה גדול יותר: ערכה ב-x = 2 או ב-x = 5?
  2. פונקציה קבועה כאשר x בין 0 ל-4, וערכה ב-x = 1 הוא 7. מהו ערכה ב-x = 3?
  3. פונקציה יורדת כאשר x בין 0 ל-10, וערכה ב-x = 4 הוא 6. האם ערכה ב-x = 8 גדול או קטן מ-6?
  4. פונקציה עולה כאשר x בין 0 ל-3, ערכה ב-x = 0 הוא 2 וב-x = 3 הוא 8. האם ערכה ב-x = 2 נמצא בין 2 ל-8?

חלק ו' — אתגרים 🏆 (שאלות 21–24)

  1. תארו גרף של פונקציה שעולה, אחר כך יורדת, ואחר כך שוב עולה. כמה נקודות מעבר יש בו?
  2. פונקציה עולה בין x = 0 ל-x = 4 ויורדת בין 4 ל-8. ידוע שערכה ב-x = 0 וב-x = 8 הוא 0, וערכה הגבוה ביותר 12. מהי הנקודה הגבוהה ביותר?
  3. האם פונקציה יכולה להיות גם עולה וגם יורדת באותו קטע עצמו?
  4. כדור נזרק מגובה 2 מטרים ומגיע לשיא של 11 מטרים אחרי 3 שניות. מה עושה הגרף אחרי x = 3?
