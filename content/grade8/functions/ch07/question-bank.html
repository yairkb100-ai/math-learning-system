חלק א' — קצב מטבלה 📋 (שאלות 1–4)

  1. כאשר x מקבל את הערכים 0, 1, 2, 3, ערכי ה-y הם 2, 6, 10, 14. מהו קצב ההשתנות?
  2. כאשר x מקבל את הערכים 0, 1, 2, 3, ערכי ה-y הם 9, 7, 5, 3. מהו קצב ההשתנות?
  3. כאשר x מקבל את הערכים 0, 2, 4, ערכי ה-y הם 0, 10, 20. מהו הקצב ליחידה אחת של x?
  4. כאשר x מקבל את הערכים 0, 1, 2, ערכי ה-y הם 3, 3, 3. מהו קצב ההשתנות?

חלק ב' — קצב משתי נקודות 📐 (שאלות 5–8)

  4. קו ישר עובר בנקודות (0, 6) ו-(6, 6). מהו קצב ההשתנות?

חלק ג' — ישר או עקומה? 📈 (שאלות 9–12)

  1. קצב ההשתנות של פונקציה קבוע. מה צורת הגרף שלה?
  2. כאשר x מקבל את הערכים 0, 1, 2, 3, ערכי ה-y הם 0, 1, 4, 9. האם הגרף קו ישר?
  3. הגרף של פונקציה הוא קו ישר. מה אפשר לומר על קצב ההשתנות שלה?
  4. כאשר x מקבל את הערכים 0, 1, 2, 3, ערכי ה-y הם 20, 17, 14, 11. האם הגרף קו ישר?

חלק ד' — חיובי או שלילי? ➕ (שאלות 13–16)

  1. בריכה מתמלאת במים. מהו סימן קצב ההשתנות של כמות המים?
  2. בריכה מתרוקנת ממים. מהו סימן קצב ההשתנות של כמות המים?
  3. נר דולק ומתקצר. מהו סימן קצב ההשתנות של גובה הנר?
  4. קצב ההשתנות של פונקציה הוא 0. מה אפשר לומר עליה?

חלק ה' — השוואת קצבים ⚖️ (שאלות 17–20)

  1. רץ א' רץ בקצב 4 קילומטרים בשעה ורץ ב' בקצב 6 קילומטרים בשעה. הגרף של מי תלול יותר?
  2. ברז א' מילא 18 ליטרים ב-6 דקות וברז ב' מילא 20 ליטרים ב-5 דקות. איזה ברז מהיר יותר?
  3. שני קווים ישרים עולים יוצאים מהראשית. איך מזהים בגרף את הקו בעל הקצב הגדול יותר?
  4. קו א' עובר בנקודות (0, 0) ו-(2, 10). קו ב' עובר בנקודות (0, 0) ו-(5, 10). איזה קו תלול יותר?

חלק ו' — אתגרים 🏆 (שאלות 21–24)

  1. פונקציה עוברת בנקודה (0, 2) וקצבה קבוע ושווה 3. מהו ערך ה-y כאשר x = 6?
  2. ערך הפונקציה ב-x = 2 הוא 7, וקצב ההשתנות קבוע ושווה −2. מהו ערכה ב-x = 6?
  3. קצב ההשתנות של פונקציה הוא 4, וערך ה-y גדל ב-24. בכמה גדל x?
  4. קו ישר בעל קצב השתנות 2 עובר בנקודות (3, a) ו-(5, 11). מהו a?
