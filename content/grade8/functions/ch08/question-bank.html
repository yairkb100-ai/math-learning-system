חלק א' — זיהוי שיפוע וחיתוך 🔍 (שאלות 1–4)

  1. מהו השיפוע של הפונקציה y = 6x − 1?
  2. באיזו נקודה חותך הישר y = −2x + 9 את ציר ה-y?
  3. מהו השיפוע של הפונקציה y = x + 4?
  4. מהם השיפוע ונקודת החיתוך של הפונקציה y = −5?

חלק ב' — חישוב ערכים 🧮 (שאלות 5–8)

  1. מהו ערך הפונקציה y = 2x + 3 עבור x = 4?
  2. מהו ערך הפונקציה y = −3x + 10 עבור x = 2?
  3. מהו ערך הפונקציה y = 5x − 7 עבור x = 0?
  4. עבור איזה x מקבלת הפונקציה y = x − 6 את הערך 0?

חלק ג' — עולה או יורד? 📈 (שאלות 9–12)

  1. האם הגרף של y = 7x − 2 עולה או יורד?
  2. האם הגרף של y = −x + 1 עולה או יורד?
  3. מה צורת הגרף של y = 8?
  4. האם הגרף של y = −6x עולה או יורד?

חלק ד' — מהטבלה אל הביטוי 📋 (שאלות 13–16)

  1. עבור x בערכים 0, 1, 2, 3 מתקבל y בערכים 2, 5, 8, 11. מהו ייצוג הפונקציה?
  2. עבור x בערכים 0, 1, 2, 3 מתקבל y בערכים 7, 5, 3, 1. מהו ייצוג הפונקציה?
  3. עבור x בערכים 0, 1, 2, 3 מתקבל y בערכים 0, 4, 8, 12. מהו ייצוג הפונקציה?
  4. עבור x בערכים 0, 1, 2, 3 מתקבל y בערכים 1, 1, 1, 1. מהו ייצוג הפונקציה?

חלק ה' — נקודה על הישר 📍 (שאלות 17–20)

  1. האם הנקודה (2, 9) נמצאת על הישר y = 4x + 1?
  2. האם הנקודה (3, 5) נמצאת על הישר y = 2x − 2?
  3. הישר y = 2x + b עובר בנקודה (1, 6). מהו b?
  4. הנקודה (5, y) נמצאת על הישר y = x + 3. מהו שיעור ה-y שלה?

חלק ו' — מקבילים ואתגרים 🏆 (שאלות 21–24)

  1. כתבו את ייצוג הישר המקביל ל-y = 3x + 2 וחותך את ציר ה-y ב-(0, −1).
  2. האם הישרים y = 4x + 1 ו-y = −4x + 1 מקבילים?
  3. ישר עובר בנקודות (0, 2) ו-(1, 6). מהו ייצוג הפונקציה?
  4. באיזו נקודה חותך הישר y = 2x − 8 את ציר ה-x?
