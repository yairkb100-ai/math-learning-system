תרגול: הפונקציה הקווית

פרק 8 · y = mx + b — שיפוע, נקודת חיתוך ושרטוט מהיר

✏️ תרגילים

נסו לפתור לבד, ורק אחר כך גלו את הפתרון.

תרגיל 1: זיהוי m ו-b קל
לכל פונקציה רשמו את השיפוע m ואת נקודת החיתוך עם ציר ה-y:

א. y = 5x - 3
ב. y = -2x + 6
ג. y = x
ד. y = 4
א. m = 5, חיתוך ב-(0, -3).
ב. m = -2, חיתוך ב-(0, 6).
ג. y = 1 · x + 0, ולכן m = 1 וחיתוך ב-(0, 0).
ד. m = 0 (ישר אופקי), חיתוך ב-(0, 4).
תרגיל 2: מהגרף אל הביטוי קל
מצאו את ייצוג הפונקציה שגרפה מופיע בשרטוט:

הישר חותך את ציר ה-y בגובה 2, ולכן b = 2. בצעד אחד ימינה הישר עולה יחידה אחת, ולכן m = 1. הייצוג: y = x + 2.
תרגיל 3: מהטבלה אל הביטוי — וחיזוי בינוני
עבור x בערכים 0, 1, 2, 3 פונקציה קווית מקבלת את הערכים 2, 6, 10, 14.

א. מצאו את ייצוג הפונקציה.
ב. חשבו את ערך הפונקציה עבור x = 10.
א. ההפרש הקבוע הוא 4, ולכן m = 4; מול x = 0 מופיע 2, ולכן b = 2. הייצוג: y = 4x + 2.
ב. y = 4 · 10 + 2 = 42.
תרגיל 4: אתגר: מקביל ובדיקת נקודה מאתגר
ישר מקביל ל-y = 3x + 1 וחותך את ציר ה-y בנקודה (0, -4).

א. מצאו את ייצוג הישר.
ב. האם הנקודה (2, 2) נמצאת על הישר? נמקו בחישוב.
א. מקבילות פירושה אותו שיפוע: m = 3; החיתוך נתון: b = -4. הייצוג: y = 3x - 4.
ב. מציבים x = 2: y = 3 · 2 - 4 = 2. התקבל בדיוק שיעור ה-y של הנקודה, ולכן (2, 2) אכן נמצאת על הישר.
תרגיל 5: זיהוי מהיר קל
מהם השיפוע ונקודת החיתוך עם ציר ה-y של הפונקציה y = -3x + 7?
השיפוע m = -3 (הישר יורד), ונקודת החיתוך עם ציר ה-y היא (0, 7).
תרגיל 6: חישוב ערך קל
מהו ערך הפונקציה y = 2x - 5 עבור x = 6?
מציבים: y = 2 · 6 - 5 = 12 - 5 = 7.
תרגיל 7: מהטבלה אל הביטוי בינוני
עבור x בערכים 0, 1, 2, 3 פונקציה קווית מקבלת את הערכים 4, 9, 14, 19. מהו ייצוג הפונקציה?
ההפרש הקבוע הוא 5, ולכן m = 5; מול x = 0 מופיע 4, ולכן b = 4. הייצוג: y = 5x + 4.
תרגיל 8: אתגר: ישר משתי נקודות מאתגר
ישר עובר בנקודות (2, 3) ו-(5, 12). מצאו את ייצוג הפונקציה.
מ-x = 2 ל-x = 5 (3 צעדים) הפונקציה עלתה מ-3 ל-12, כלומר ב-9 — שלוש יחידות לכל צעד: m = 3. מציבים את (2, 3): 3 = 3 · 2 + b, ולכן b = -3. הייצוג: y = 3x - 3 (בדיקה: 3 · 5 - 3 = 12).
