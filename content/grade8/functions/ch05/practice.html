תרגול: שרטוט גרפים

פרק 5 · מטבלת ערכים לנקודות, מנקודות לקו ישר

✏️ תרגילים

נסו לפתור לבד, ורק אחר כך גלו את הפתרון.

תרגיל 1: טבלה לפונקציה y = 2x קל
השלימו טבלת ערכים לפונקציה y = 2x עבור x = -2, -1, 0, 1, 2, ורשמו את חמשת הזוגות הסדורים.
מציבים כל ערך:
x = -2: y = -4
x = -1: y = -2
x = 0: y = 0
x = 1: y = 2
x = 2: y = 4

הזוגות: (-2, -4), (-1, -2), (0, 0), (1, 2), (2, 4).
תרגיל 2: הפונקציה y = x − 3 קל
בנו טבלת ערכים לפונקציה y = x - 3 עבור x = -1, 0, 1, 2, 3. באיזו נקודה הגרף חותך את ציר ה-x?
ערכי y לפי הסדר: -4, -3, -2, -1, 0. הגרף חותך את ציר ה-x כאשר y = 0, וזה קורה בנקודה (3, 0).
תרגיל 3: שרטוט קו יורד בינוני
שרטטו את הגרף של הפונקציה y = -x + 2: בנו טבלת ערכים עבור x = -2, -1, 0, 1, 2, סמנו את הנקודות וחברו אותן.

ערכי y לפי הסדר: 4, 3, 2, 1, 0. הנקודות (-2, 4), (-1, 3), (0, 2), (1, 1), (2, 0) מסתדרות על קו ישר יורד: ככל ש-x גדל ב-1, y קטן ב-1. הקו חותך את ציר ה-y בנקודה (0, 2) ואת ציר ה-x בנקודה (2, 0).
תרגיל 4: אתגר: מהנקודות אל החוק ובחזרה מאתגר
גרף של פונקציה מהצורה y = mx + b עובר דרך הנקודות (0, -1) ו-(1, 1).

א. מצאו את הביטוי האלגברי של הפונקציה.
ב. בדקו בעזרת הצבה האם הנקודה (4, 7) נמצאת על הגרף.
ג. מהו ערך y כאשר x = -2?
א. מ-(0, -1) נובע b = -1. מ-(0, -1) ל-(1, 1) ערך y גדל ב-2 כאשר x גדל ב-1, ולכן m = 2. הפונקציה: y = 2x - 1.
ב. מציבים: 2 · 4 - 1 = 7 — כן, הנקודה על הגרף.
ג. y = 2 · (-2) - 1 = -5.
תרגיל 5: טבלה מהירה קל
בנו טבלת ערכים לפונקציה y = 3x עבור x = -2, -1, 0, 1, 2.
ערכי y לפי הסדר: -6, -3, 0, 3, 6. כל הנקודות על קו ישר שעובר דרך הראשית.
תרגיל 6: חיתוך עם ציר ה-y קל
באיזו נקודה חותך הגרף של y = 4x + 3 את ציר ה-y?
מציבים x = 0 ומקבלים y = 3. נקודת החיתוך: (0, 3).
תרגיל 7: עולה או יורד? בינוני
בלי לשרטט, קבעו לגבי כל פונקציה האם הגרף שלה עולה או יורד משמאל לימין: א. y = 5x - 2 ב. y = -3x + 1.
א. המקדם של x חיובי (5): בכל צעד ימינה y גדל — הקו עולה.
ב. המקדם של x שלילי (-3): בכל צעד ימינה y קטן — הקו יורד.
תרגיל 8: אתגר: פגישה של שני קווים מאתגר
באיזו נקודה נפגשים הגרפים של y = 3x - 2 ושל y = x + 4? בדקו את תשובתכם בהצבה בשתי הפונקציות.
בנקודת המפגש שתי הפונקציות נותנות אותו y: 3x - 2 = x + 4, ולכן 2x = 6 ו-x = 3. מציבים: y = 3 · 3 - 2 = 7 וגם 3 + 4 = 7. נקודת המפגש: (3, 7).
