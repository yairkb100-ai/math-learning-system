חלק א' — כלל החזקה וליניאריות 🎯 (שאלות 1–5)

  1. גזרו: f(x) = x⁷.
  2. גזרו: f(x) = 5x³ − 2x.
  3. גזרו: f(x) = 3x⁴ − x² + 8.
  4. גזרו: f(x) = 6x⁵ − 4x³ + 2x − 1.
  5. גזרו: f(x) = 10x − 7.

חלק ב' — כלל המכפלה 🔗 (שאלות 6–10)

  1. גזרו: y = (x + 2)(x² + 1).
  2. גזרו: y = (3x − 1)(x + 4).
  3. גזרו: y = x²(2x − 3).
  4. גזרו ובדקו בפתיחת סוגריים: y = (x + 1)(x − 1).
  5. נכון/לא נכון: (f·g)′ = f′·g′. נמקו.

חלק ג' — כלל המנה ➗ (שאלות 11–14)

  1. גזרו: y = x / (x + 1).
  2. גזרו: y = (x + 1) / (x − 1).
  3. גזרו: y = x² / (x² + 1).
  4. גזרו: y = (2x) / (x² + 3).

חלק ד' — כלל השרשרת ⛓️ (שאלות 15–19)

  1. גזרו: y = (x² + 1)³.
  2. גזרו: y = (2x + 3)⁵.
  3. גזרו: y = (x³ − x)⁴.
  4. גזרו: y = (5 − x²)⁶.
  5. מהי הטעות הנפוצה ביותר בכלל השרשרת?

חלק ה' — נגזרת בנקודה ואתגר 🔥 (שאלות 20–24)

  1. מצאו f′(1) עבור f(x) = x⁴ − 2x².
  2. מצאו f′(3) עבור f(x) = x³ − 5x.
  3. מצאו את משוואת המשיק ל-f(x) = x² ב-x = 2.
  4. פשטו וגזרו: y = (x⁴ + x²) / x².
  5. מצאו f′(0) עבור f(x) = (x + 1)³.
