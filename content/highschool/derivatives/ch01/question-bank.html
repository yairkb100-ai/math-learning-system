24 שאלות · חשבון דיפרנציאלי · פרק 1 — מיתר ומשיק, גבול, הנגזרת מן ההגדרה ומשוואת המשיק · דף פתרונות בעמוד האחרון
שם: כיתה: תאריך:

חלק א' — גבולות ומנת ההפרשים 🎯 (שאלות 1–5)

  1. חשבו: lim(h→0) (7 + h).
  2. חשבו: lim(h→0) (2x + h + 1).
  3. חשבו: lim(h→0) (h² − 5h + 2).
  4. צמצמו את h: (3xh + h²) / h.
  5. עבור f(x) = x² כתבו את מנת ההפרשים המצומצמת.

חלק ב' — נגזרת מן ההגדרה 🔧 (שאלות 6–10)

  1. מצאו מן ההגדרה את הנגזרת של f(x) = 4x.
  2. מצאו מן ההגדרה את הנגזרת של f(x) = x² − x.
  3. מצאו מן ההגדרה את f′(2) עבור f(x) = x².
  4. מצאו מן ההגדרה את הנגזרת של f(x) = 3x².
  5. מהי הנגזרת של פונקציה קבועה f(x) = 9?

חלק ג' — שיפוע ומשמעות 📐 (שאלות 11–15)

  1. ידוע (x²)′ = 2x. מהו שיפוע המשיק ל-y = x² ב-x = 7?
  2. באיזו נקודה שיפוע המשיק ל-y = x² שווה ל-12?
  3. מקום גוף s(t) = t², מהירות s′(t) = 2t. מהי המהירות ב-t = 4?
  4. מה משמעות f′(a) מבחינה גיאומטרית?
  5. היכן המשיק לגרף y = x² אופקי?

חלק ד' — משוואת המשיק 📏 (שאלות 16–19)

  1. מצאו את משוואת המשיק ל-f(x) = x² ב-x = 2.
  2. מצאו את משוואת המשיק ל-f(x) = x² ב-x = −1.
  3. מצאו את משוואת המשיק ל-f(x) = x² ב-x = 0.
  4. מהי משוואת המשיק לפונקציה קווית f(x) = 3x + 1 בכל נקודה?

חלק ה' — אתגר 🔥 (שאלות 20–24)

  1. מצאו מן ההגדרה את הנגזרת של f(x) = 1/x.
  2. מצאו מן ההגדרה את הנגזרת של f(x) = x³ (רמז: (x+h)³ = x³ + 3x²h + 3xh² + h³).
  3. נכון/לא נכון: מותר להציב h = 0 במנת ההפרשים לפני צמצום. נמקו.
  4. עבור f(x) = x² מצאו את הנקודה שבה שיפוע המשיק −4.
  5. אם f′(a) = 0, מה אפשר לומר על המשיק בנקודה (a, f(a))?

1. 7.
2. 2x + 1.
3. 2 (האיברים עם h מתאפסים).
4. 3x + h → כאשר h → 0: 3x.
5. 2x + h (ולפני ההשאפה h → 0 נותן 2x).
6. (4(x+h) − 4x)/h = 4.
7. ( (x+h)² − (x+h) − x² + x )/h = 2x + h − 1 → 2x − 1.
8. ( (2+h)² − 4 )/h = (4h + h²)/h = 4 + h → f′(2) = 4.
9. (3(x+h)² − 3x²)/h = (6xh + 3h²)/h = 6x + 3h → 6x.
10. 0 (מנת ההפרשים 0/h = 0).
11. 2 · 7 = 14.
12. 2x = 12 → x = 6.
13. s′(4) = 8 מ׳/ש׳.
14. f′(a) הוא שיפוע המשיק לגרף בנקודה (a, f(a)).
15. ב-x = 0 (הקדקוד), שם f′(0) = 0.
16. f(2) = 4, f′(2) = 4: y = 4x − 4.
17. f(−1) = 1, f′(−1) = −2: y = −2x − 1.
18. f(0) = 0, f′(0) = 0: y = 0 (ציר ה-x).
19. y = 3x + 1 — המשיק לישר הוא הישר עצמו (שיפוע קבוע 3).
20. f′(x) = −1/x².
21. ( (x+h)³ − x³ )/h = (3x²h + 3xh² + h³)/h = 3x² + 3xh + h² → 3x².
22. לא נכון — זו חלוקה באפס. מצמצמים h ורק אז משאיפים h → 0.
23. 2x = −4 → x = −2, הנקודה (−2, 4).
24. המשיק אופקי (מקביל לציר ה-x); זו נקודה קריטית — מועמדת לקיצון.
