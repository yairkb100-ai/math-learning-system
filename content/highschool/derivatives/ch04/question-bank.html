מאגר שאלות — פרק 4: בעיות קיצון (אופטימיזציה)

25 שאלות · חשבון דיפרנציאלי · פרק 4 — פונקציית מטרה, גזירה ומיצוי בעיות מהעולם האמיתי · דף פתרונות בעמוד האחרון
שם: כיתה: תאריך:

חלק א' — מכפלה וסכום 🎯 (שאלות 1–5)

  1. מבין הזוגות שסכומם 12, מצאו את בעל המכפלה המקסימלית.
  2. מבין הזוגות שסכומם 30, מהי המכפלה המקסימלית?
  3. סכום שני מספרים 14. מצאו את מכפלתם המקסימלית.
  4. סכום שני מספרים 8. מצאו את המספרים שעבורם סכום ריבועיהם מינימלי.
  5. מכפלת שני מספרים שסכומם s מקסימלית כאשר הם שווים. נכון או לא נכון?

חלק ב' — שטח והיקף 📐 (שאלות 6–10)

  1. היקף מלבן 20 ס"מ. מהו השטח המקסימלי?
  2. היקף מלבן 40 מ'. אילו מידות נותנות שטח מקסימלי?
  3. מלבן צמוד לקיר בצלע אחת, סך הגדר 40 מ'. מצאו שטח מקסימלי.
  4. היקף מלבן 28 ס"מ. מהן מידות השטח המקסימלי?
  5. מבין כל המלבנים בהיקף נתון, לאיזה השטח הגדול ביותר?

חלק ג' — בניית פונקציית מטרה 🏗️ (שאלות 11–15)

  1. כתבו פונקציית מטרה למכפלת שני מספרים שסכומם 25.
  2. כתבו פונקציית שטח למלבן שהיקפו 50, צלע x.
  3. כתבו פונקציית שטח לערוגה צמודה לקיר עם 80 מ' גדר, צלע ניצבת x.
  4. מהו השלב הראשון בפתרון בעיית קיצון?
  5. מדוע יש לבטא את פונקציית המטרה במשתנה אחד לפני הגזירה?

חלק ד' — מיצוי מלא 🔬 (שאלות 16–20)

  1. מצאו את המקסימום של S(x) = x(40 − 2x) עבור 0 ≤ x ≤ 20.
  2. מצאו את המינימום של f(x) = x² + (12 − x)².
  3. מצאו את המקסימום של P(x) = x(18 − x).
  4. מצאו את המידות של מלבן בהיקף 32 עם שטח מקסימלי.
  5. מצאו את המקסימום של f(x) = x(60 − 3x).

חלק ה' — אתגר 🔥 (שאלות 21–25)

  1. מלבן ששטחו 25 סמ"ר; מצאו את ההיקף המינימלי (רמז: P = 2x + 50/x).
  2. מבין הזוגות שהפרשם 6, מצאו את בעל המכפלה המינימלית.
  3. נכון/לא נכון: בתחום סגור הקיצון תמיד בנקודה קריטית. נמקו.
  4. לחקלאי 100 מ' גדר לשתי ערוגות מלבניות זהות זו לצד זו (3 קטעי גדר לרוחב). זהו את המשתנה ובנו פונקציית שטח.
  5. מדוע פתרון שנותן אורך שלילי נפסל בבעיית קיצון?

🔑 פתרונות מלאים — מאגר שאלות פרק 4

למורה / להורה / לבדיקה עצמית
1. P = x(12 − x) → x = 6, מכפלה מקסימלית 36.
2. x = 15, מכפלה מקסימלית 225.
3. x = 7, מכפלה מקסימלית 49.
4. f = x² + (8−x)² → x = 4; המספרים 4 ו-4, סכום ריבועים 32.
5. נכון — המקסימום כאשר שני המספרים s/2.
6. ריבוע 5×5, שטח מקסימלי 25 סמ"ר.
7. ריבוע 10×10 מ׳, שטח 100 מ"ר.
8. S = x(40 − 2x) → x = 10, מידות 10×20, שטח מקסימלי 200 מ"ר.
9. ריבוע 7×7, שטח מקסימלי 49 סמ"ר.
10. לריבוע.
11. P(x) = x(25 − x).
12. S(x) = x(25 − x) (צלע שנייה 25 − x).
13. S(x) = x(80 − 2x).
14. הגדרת משתנה ובניית פונקציית מטרה במשתנה אחד.
15. כי הנגזרת מוגדרת לגבי משתנה יחיד — אי אפשר לגזור ולמצוא קיצון בשני משתנים בלתי תלויים.
16. S′ = 40 − 4x = 0 → x = 10, מקסימום S = 200 (בקצוות 0 ו-0).
17. f′ = 4x − 24 = 0 → x = 6; מינימום 72 (המספרים 6 ו-6).
18. x = 9, מקסימום 81.
19. ריבוע 8×8, שטח מקסימלי 64.
20. f′ = 60 − 6x = 0 → x = 10, מקסימום f = 10·30 = 300.
21. P′ = 2 − 50/x² = 0 → x = 5; ריבוע 5×5, היקף מינימלי 20 ס"מ.
22. P = x(x − 6); P′ = 2x − 6 = 0 → x = 3; המספרים 3 ו-−3, מכפלה מינימלית −9.
23. לא נכון — הקיצון עשוי להתקבל בקצה התחום, ולכן בודקים גם קצוות.
24. משתנה x = צלע משותפת; 3x + 2y = 100 → y = (100 − 3x)/2; שטח S = 2x · y = x(100 − 3x).
25. כי גודל פיזי (אורך/זמן) אינו יכול להיות שלילי — הפתרון אינו הגיוני במציאות.
