דף עבודה — פרק 3: חקירת פונקציות: עלייה, ירידה וקיצון

חשבון דיפרנציאלי · פרק 3 — סימן הנגזרת, נקודות קריטיות וטבלת סימנים
שם: כיתה: תאריך:
אם f′(x) > 0 — הפונקציה עולה; אם f′(x) < 0 — יורדת; אם f′(x) = 0 — נקודה קריטית. מעבר מ-+ ל-−: מקסימום; מ-− ל-+: מינימום; אין החלפת סימן — אין קיצון. את ערך ה-y מציבים ב-f, לא ב-f′.

שאלה 1 — נקודות קריטיות קל

גזרו ופתרו f′(x) = 0 כדי למצוא את הנקודות הקריטיות:

  1. f(x) = x² + 6x
  2. f(x) = x² − 8x
  3. f(x) = x³ − 12x

שאלה 2 — מינימום של פרבולה קל

מצאו את נקודת הקיצון וקבעו את סוגה (מקסימום/מינימום), כולל ערך ה-y:

  1. f(x) = x² + 6x
  2. f(x) = −x² + 4x

שאלה 3 — טבלת סימנים — מעוקבת בינוני

עבור f(x) = x³ − 12x: גזרו, מצאו נקודות קריטיות, בנו טבלת סימנים וקבעו את סוג כל קיצון (כולל ערכי y).

שאלה 4 — תחומי עלייה וירידה בינוני

מצאו את תחומי העלייה והירידה ואת נקודות הקיצון של f(x) = x³ − 3x².

שאלה 5 — כיוון לפי סימן הנגזרת קל

נתונה f′(x) = 2x − 4. באילו x הפונקציה עולה, ובאילו יורדת? מהי הנקודה הקריטית?

שאלה 6 — נקודה קריטית שאינה קיצון מאתגר

עבור f(x) = x³: הראו ש-f′(0) = 0 אך אין שם קיצון (בדקו את סימן f′ משני צידי האפס). איזה סוג נקודה זו?

שאלה 7 — חקירת מעלה 4 מאתגר

מצאו את נקודות הקיצון של f(x) = x⁴ − 4x³ (רמז: f′(x) = 4x²(x − 3) — שימו לב לנקודה שבה אין החלפת סימן).

שאלה 8 — מהו סוג הקיצון בינוני

לכל מעבר בטבלת הסימנים כתבו איזה קיצון מתקבל:

  1. f′ עוברת מ-+ ל-− ⟵
  2. f′ עוברת מ-− ל-+ ⟵
  3. f′ אינה מחליפה סימן ⟵

שאלה 9 — מגרף הנגזרת לפונקציה בינוני

נתון ש-f′(x) > 0 עבור x < 1 ו-f′(x) < 0 עבור x > 1. מה קורה לפונקציה f סביב x = 1? איזה סוג נקודה יש שם?

שאלה 10 — אתגר — קיצון עם פרמטר מאתגר

נתונה f(x) = x² − 2ax. הראו שנקודת הקיצון היא x = a, וקבעו אם זהו מקסימום או מינימום (לפי סימן המקדם המוביל).

שאלה 2. א. x = −3, f′ מ-− ל-+ → מינימום (−3, −9)   ב. x = 2, f′ = −2x + 4 מ-+ ל-− → מקסימום (2, 4)

שאלה 4. f′ = 3x² − 6x = 3x(x − 2) → x = 0, 2. עולה ב-x<0 וב-x>2; יורדת ב-0. מקסימום (0, 0), מינימום (2, −4).
שאלה 5. 2x − 4 > 0 → עולה עבור x > 2; יורדת עבור x < 2. נקודה קריטית x = 2 (מינימום).
שאלה 6. f′ = 3x². f′(0) = 0, אבל 3x² > 0 משני הצדדים — אין החלפת סימן, אין קיצון. זו נקודת פיתול עם משיק אופקי.
שאלה 7. f′ = 4x³ − 12x² = 4x²(x − 3) → x = 0, 3. סביב x = 0 אין החלפת סימן (4x² ≥ 0) → לא קיצון. ב-x = 3 מ-− ל-+ → מינימום (3, −27).
שאלה 8. א. מקסימום   ב. מינימום   ג. אין קיצון
שאלה 9. הפונקציה עולה עד x = 1 ואז יורדת → מקסימום מקומי ב-x = 1.
שאלה 10. f′ = 2x − 2a = 0 → x = a. המקדם המוביל חיובי → פרבולה מחייכת → מינימום ב-x = a.
