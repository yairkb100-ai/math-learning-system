תרגול: חקירת פונקציות: עלייה, ירידה וקיצון

פרק 3 · חשבון דיפרנציאלי · פרק 3 — סימן הנגזרת, נקודות קריטיות וטבלת סימנים

✏️ תרגילים

נסו לפתור לבד, ורק אחר כך גלו את הפתרון.

תרגיל 1: מינימום של פרבולה קל
מצאו את נקודת הקיצון של f(x) = x^2 + 6x וקבעו את סוגה.
f'(x) = 2x + 6 = 0 \Rightarrow x = -3. הנגזרת שלילית משמאל וחיובית מימין — מינימום. f(-3) = 9 - 18 = -9. נקודת מינימום: (-3, -9).
תרגיל 2: חקירת פונקציה מעוקבת בינוני
מצאו את נקודות הקיצון של f(x) = x^3 - 12x וקבעו את סוגן.
f'(x) = 3x^2 - 12 = 0 \Rightarrow x = \pm 2.

טבלת סימנים: f' > 0 עבור x < -2, f' < 0 בין -2 ל-2, f' > 0 עבור x > 2.

מקסימום מקומי (-2, 16), מינימום מקומי (2, -16).
תרגיל 3: תחומי עלייה וירידה בינוני
מצאו את תחומי העלייה והירידה של f(x) = x^3 - 3x^2 ואת נקודות הקיצון.
f'(x) = 3x^2 - 6x = 3x(x - 2) = 0 \Rightarrow x = 0, 2.

f' > 0 עבור x < 0 או x > 2 (עלייה); f' < 0 עבור 0 < x < 2 (ירידה).

מקסימום מקומי: f(0) = 0, כלומר (0, 0). מינימום מקומי: f(2) = 8 - 12 = -4, כלומר (2, -4).
תרגיל 4: נקודה קריטית כפולה מאתגר
מצאו את נקודות הקיצון של f(x) = x^4 - 4x^3.
f'(x) = 4x^3 - 12x^2 = 4x^2(x - 3) = 0 \Rightarrow x = 0, 3.

סימן הנגזרת: עבור x < 0 הביטוי 4x^2 > 0 ו-(x-3) < 0, ולכן f' < 0; עבור 0 < x < 3 עדיין f' < 0; עבור x > 3: f' > 0.

ב-x = 0 אין החלפת סימן — לא קיצון. ב-x = 3 מעבר מ-- ל-+ — מינימום מקומי, f(3) = 81 - 108 = -27, כלומר (3, -27).
