✏️ תרגילים

נסו לפתור לבד, ורק אחר כך גלו את הפתרון.

תרגיל 1: זיהוי מקדמים קל
כתוב את המקדמים a, b, c של המשוואה 3x^2 - 7x + 2 = 0.
a = 3, b = -7, c = 2. שימו לב שהסימן המינוס שייך למקדם, ולכן b = -7.
תרגיל 2: סידור לצורה הכללית קל
סדר את המשוואה 2x^2 = 5x - 3 לצורה הכללית וזהה את המקדמים.
מעבירים הכל לאגף אחד: 2x^2 - 5x + 3 = 0. המקדמים: a = 2, b = -5, c = 3.
תרגיל 3: פתיחת סוגריים בינוני
כתוב את המשוואה (x - 2)(x + 3) = 0 בצורה הכללית.
פותחים סוגריים: x^2 + 3x - 2x - 6 = x^2 + x - 6 = 0. אז a = 1, b = 1, c = -6.
תרגיל 4: בדיקת פתרון בינוני
בדוק אם x = 3 הוא פתרון של המשוואה x^2 - 5x + 6 = 0.
מציבים: 3^2 - 5 · 3 + 6 = 9 - 15 + 6 = 0. התקבל אפס, ולכן x = 3 הוא פתרון.
תרגיל 5: מציאת מקדם חסר מאתגר
עבור איזה ערך של c המספר x = 3 הוא פתרון של x^2 - 4x + c = 0?
מציבים x = 3: 3^2 - 4 · 3 + c = 9 - 12 + c = -3 + c = 0, ולכן c = 3.
