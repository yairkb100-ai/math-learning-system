מאגר שאלות — פרק 1: הצורה הכללית וזיהוי מקדמים

31 שאלות · משוואות ריבועיות · פרק 1 — הצורה הכללית, המקדמים a, b, c ובדיקת פתרון · דף פתרונות בעמוד האחרון
שם: כיתה: תאריך:

חלק א' — זיהוי מקדמים 🔎 (שאלות 1–6)

  1. זהו את המקדמים a, b, c של x² − 8x + 15 = 0.
  2. זהו את המקדמים של 3x² + 2x − 5 = 0.
  3. זהו את המקדמים של −2x² + 7x − 1 = 0.
  4. זהו את המקדמים של x² − 36 = 0 (שימו לב ל-b).
  5. זהו את המקדמים של 6x² + x = 0 (שימו לב ל-c).
  6. במשוואה x² − 5x + 6 = 0, מהו הערך של b? (עם הסימן)

חלק ב' — סידור לצורה הכללית 📐 (שאלות 7–12)

  1. סדרו לצורה הכללית וזהו מקדמים: 3x² = 7x − 2.
  2. סדרו לצורה הכללית: x² + 10 = 7x.
  3. סדרו לצורה הכללית: 5x² = 20.
  4. סדרו לצורה הכללית: 2x(x − 3) = 0.
  5. כתבו בצורה הכללית: (x − 4)(x + 1) = 0.
  6. כתבו בצורה הכללית: (2x + 3)(x − 2) = 0.

חלק ג' — בדיקת פתרון בהצבה ✅ (שאלות 13–17)

  1. בדקו אם x = 5 פותר את x² − 8x + 15 = 0.
  2. בדקו אם x = −3 פותר את x² + x − 6 = 0.
  3. בדקו אם x = 1 פותר את 2x² − 3x + 1 = 0.
  4. אילו מבין 1, 2, 3 הם פתרונות של x² − 5x + 6 = 0?
  5. המספר x = 0 הוא פתרון של x² − 4x = 0? נמקו.

חלק ד' — טיפוסים ופתרון מהיר ⚡ (שאלות 18–22)

  1. פתרו בהוצאת גורם משותף: x² − 6x = 0.
  2. פתרו בהוצאת גורם משותף: 2x² + 8x = 0.
  3. פתרו בהוצאת שורש: x² − 16 = 0.
  4. פתרו בהוצאת שורש: 3x² − 12 = 0.
  5. לאיזה טיפוס שייכת x² − 100 = 0 — מלאה / בלי b / בלי c?

חלק ה' — פרבולה ומספר פתרונות 📊 (שאלות 23–26)

  1. הפרבולה y = x² − 4x + 3 — מחייכת או עצובה? נמקו לפי a.
  2. פרבולה חותכת את ציר ה-x בשתי נקודות. כמה שורשים למשוואה?
  3. פרבולה נוגעת בציר ה-x בנקודה אחת. כמה שורשים?
  4. כתבו את שלוש האפשרויות למספר נקודות החיתוך של פרבולה עם ציר ה-x.

חלק ו' — אתגר 🔥 (שאלות 27–31)

  1. עבור איזה c המספר x = 2 פותר את x² − 5x + c = 0?
  2. ידוע ש-x = 3 פותר את x² + bx − 6 = 0. מצאו b.
  3. בנו משוואה ריבועית ששורשיה x = −1 ו-x = 4 (בצורה הכללית).
  4. עבור אילו ערכי m המשוואה (m − 2)x² + 3x + 1 = 0 אינה ריבועית?
  5. נכון/לא נכון: אם c = 0 אז x = 0 תמיד פתרון של ax² + bx + c = 0. נמקו.

13. כן: 25 − 40 + 15 = 0 ✓.
14. כן: 9 − 3 − 6 = 0 ✓.
15. כן: 2 − 3 + 1 = 0 ✓.
16. מציבים: x = 2: 4 − 10 + 6 = 0 ✓ ו-x = 3: 9 − 15 + 6 = 0 ✓. x = 1: 1 − 5 + 6 = 2 ≠ 0. השורשים: 2 ו-3.
17. כן: 0 − 0 = 0 ✓. כאשר אין איבר חופשי, x = 0 תמיד פתרון.
18. x(x − 6) = 0 → x = 0 או x = 6.
19. 2x(x + 4) = 0 → x = 0 או x = −4.
20. x² = 16 → x = 4 או x = −4.
21. x² = 4 → x = 2 או x = −2.
22. בלי b (אין איבר x): a = 1, b = 0, c = −100.
23. מחייכת, כי a = 1 > 0 (פתוחה כלפי מעלה).
24. שני שורשים.
25. שורש יחיד (פתרון כפול).
26. שתי נקודות (2 שורשים), נקודה אחת (שורש יחיד), או אף נקודה (אין פתרון ממשי).
27. x = 2: 4 − 10 + c = 0 → c = 6.
28. x = 3: 9 + 3b − 6 = 0 → 3b = −3 → b = −1.
29. (x + 1)(x − 4) = 0 → x² − 3x − 4 = 0.
30. כאשר m − 2 = 0, כלומר m = 2 — אז נשאר 3x + 1 = 0, לא ריבועית.
31. נכון. אם c = 0 אז ax² + bx = x(ax + b) = 0, ולכן x = 0 מאפס את המשוואה.
