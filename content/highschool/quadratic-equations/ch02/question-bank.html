חלק ד' — דיסקרימיננט ומספר פתרונות 🔬 (שאלות 16–20)

  1. חשבו Δ וקבעו מספר פתרונות: x² − 3x + 2 = 0.
  2. חשבו Δ וקבעו מספר פתרונות: x² + 2x + 7 = 0.
  3. חשבו Δ וקבעו מספר פתרונות: 9x² − 6x + 1 = 0.
  4. מהו מספר הפתרונות כאשר Δ = 0? כיצד נראית הפרבולה?
  5. מהו מספר הפתרונות כאשר Δ < 0? כיצד נראית הפרבולה?

חלק ה' — אתגר ופרמטרים 🔥 (שאלות 21–25)

  1. עבור אילו ערכי k למשוואה x² − 4x + k = 0 יש פתרון יחיד?
  2. עבור אילו ערכי c למשוואה x² + 6x + c = 0 אין פתרון ממשי?
  3. הראו שלמשוואה x² + 1 = 0 אין פתרון ממשי.
  4. המשוואה x² − 2mx + 9 = 0 בעלת פתרון יחיד. מצאו את m (שני ערכים).
  5. נכון/לא נכון: אם Δ הוא ריבוע מושלם, שורשי המשוואה (עם מקדמים שלמים ו-a=1) הם מספרים שלמים. נמקו.

🔑 פתרונות מלאים — מאגר שאלות פרק 2

למורה / להורה / לבדיקה עצמית
1. (x − 2)(x − 4) = 0 → x = 2 או x = 4.
2. (x + 2)(x + 5) = 0 → x = −2 או x = −5.
3. (x − 4)(x + 3) = 0 → x = 4 או x = −3.
4. x = 6 או x = −1.
5. (x − 4)(x − 7) = 0 → x = 4 או x = 7.
6. (x + 6)(x − 3) = 0 → x = −6 או x = 3.
7. (x + 4)² = 9 → x = −1 או x = −7.
8. (x − 3)² = 1 → x = 4 או x = 2.
9. (x + 2)² = 16 → x = 2 או x = −6.
10. חצי מ-10 הוא 5, בריבוע 25. מוסיפים 25: x² + 10x + 25 = (x + 5)².
11. Δ = 4 + 32 = 36, x = (2 ± 6)/2 → x = 4 או x = −2.
12. Δ = 0, x = −3 (פתרון יחיד).
13. Δ = 49 − 24 = 25, x = (7 ± 5)/4 → x = 3 או x = 1/2.
14. Δ = 4 + 12 = 16, x = (−2 ± 4)/6 → x = 1/3 או x = −1.
15. Δ = 12, x = (4 ± 2√3)/2 = 2 ± √3.
16. Δ = 9 − 8 = 1 > 0 → שני פתרונות (x = 1, 2).
17. Δ = 4 − 28 = −24 < 0 → אין פתרון ממשי.
18. Δ = 36 − 36 = 0 → פתרון יחיד (x = 1/3).
19. פתרון יחיד (כפול); הפרבולה נוגעת בציר ה-x בנקודה אחת (הקדקוד).
20. אין פתרון ממשי; הפרבולה אינה פוגשת את ציר ה-x כלל.
21. Δ = 16 − 4k = 0 → k = 4.
22. Δ = 36 − 4c < 0 → c > 9.
23. Δ = 0 − 4 = −4 < 0, או ישירות x² = −1 — אין מספר ממשי שריבועו שלילי.
24. Δ = 4m² − 36 = 0 → m² = 9 → m = 3 או m = −3.
25. נכון. אם Δ ריבוע מושלם, √Δ שלם, ובנוסחה x = (−b ± √Δ)/2 מתקבלים ערכים רציונליים/שלמים.
