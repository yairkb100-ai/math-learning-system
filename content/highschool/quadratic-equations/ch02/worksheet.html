דף עבודה — פרק 2: שיטות פתרון והדיסקרימיננט

משוואות ריבועיות · פרק 2 — פירוק לגורמים, השלמה לריבוע, נוסחת השורשים והדיסקרימיננט Δ
שם: כיתה: תאריך:
נוסחת השורשים: x = (−b ± √(b² − 4ac)) / 2a. הדיסקרימיננט: Δ = b² − 4ac. זכרו — Δ > 0: שני פתרונות, Δ = 0: פתרון יחיד, Δ < 0: אין פתרון ממשי.

שאלה 1 — פירוק לגורמים — חימום קל

מצאו שני מספרים שמכפלתם היא האיבר החופשי וסכומם הוא −b, ופתרו:

  1. x² − 7x + 12 = 0
  2. x² − 5x + 6 = 0
  3. x² + 5x + 6 = 0
  4. x² − x − 6 = 0

שאלה 2 — פירוק לגורמים — סימנים מעורבים בינוני

פתרו בפירוק לגורמים:

  1. x² + 2x − 15 = 0
  2. x² − 9x + 20 = 0
  3. x² − 3x − 10 = 0

שאלה 3 — מכפלה שווה אפס קל

פתרו ישירות (המכפלה כבר מפורקת):

  1. (x − 3)(x − 4) = 0
  2. (x + 2)(x − 7) = 0
  3. x(x − 5) = 0

שאלה 4 — השלמה לריבוע בינוני

פתרו בשיטת השלמה לריבוע (הראו את כל השלבים):

  1. x² + 6x + 5 = 0
  2. x² − 4x − 5 = 0
  3. x² + 2x − 8 = 0

שאלה 5 — חישוב הדיסקרימיננט קל

חשבו את Δ = b² − 4ac וקבעו כמה פתרונות ממשיים יש (בלי לפתור):

  1. x² − 5x + 6 = 0
  2. x² − 4x + 4 = 0
  3. x² + x + 1 = 0
  4. 2x² − 3x + 5 = 0

שאלה 6 — נוסחת השורשים — a=1 בינוני

פתרו בעזרת נוסחת השורשים:

  1. x² + 2x − 3 = 0
  2. x² − 6x + 8 = 0
  3. x² + 4x + 4 = 0

שאלה 7 — נוסחת השורשים — a≠1 מאתגר

פתרו בעזרת נוסחת השורשים (שימו לב למקדם המוביל):

  1. 2x² − 4x − 6 = 0
  2. 3x² − 5x − 2 = 0
  3. 2x² + 7x + 3 = 0

שאלה 8 — בוחרים שיטה בינוני

לכל משוואה כתבו איזו שיטה נוחה ביותר (פירוק / נוסחה) ואז פתרו:

  1. x² − 8x + 15 = 0
  2. x² − 2x − 4 = 0

שאלה 9 — דיסקרימיננט ופרבולה קל

לכל ערך של Δ, כתבו כמה פתרונות ואיך הפרבולה מתנהגת מול ציר ה-x:

  1. Δ = 9 ⟵
  2. Δ = 0 ⟵
  3. Δ = −5 ⟵

שאלה 10 — פתרון כפול בינוני

הראו שהמשוואה x² − 10x + 25 = 0 היא בעלת פתרון יחיד: חשבו את Δ, מצאו את הפתרון, וכתבו את המשוואה כריבוע (x − ...)².

שאלה 11 — כמה פתרונות — מציאת פרמטר מאתגר

עבור אילו ערכים של c למשוואה x² − 6x + c = 0 יש פתרון יחיד? (רמז: Δ = 0.)

שאלה 12 — אתגר — אין פתרון מאתגר

הראו שלמשוואה x² + 2x + 5 = 0 אין פתרון ממשי, והסבירו במילים מה זה אומר על הפרבולה y = x² + 2x + 5.

שאלה 1. א. (x − 3)(x − 4) = 0 → x = 3, 4   ב. (x − 2)(x − 3) = 0 → x = 2, 3   ג. (x + 2)(x + 3) = 0 → x = −2, −3   ד. (x − 3)(x + 2) = 0 → x = 3, −2
שאלה 2. א. (x + 5)(x − 3) = 0 → x = −5, 3   ב. (x − 4)(x − 5) = 0 → x = 4, 5   ג. (x − 5)(x + 2) = 0 → x = 5, −2
שאלה 3. א. x = 3 או x = 4   ב. x = −2 או x = 7   ג. x = 0 או x = 5
שאלה 4. א. (x + 3)² = 4 → x = −1, −5   ב. (x − 2)² = 9 → x = 5, −1   ג. (x + 1)² = 9 → x = 2, −4
שאלה 5. א. Δ = 25 − 24 = 1 > 0 → 2 פתרונות   ב. Δ = 16 − 16 = 0 → פתרון יחיד   ג. Δ = 1 − 4 = −3 < 0 → אין פתרון   ד. Δ = 9 − 40 = −31 < 0 → אין פתרון
שאלה 6. א. Δ = 16, x = (−2 ± 4)/2 → x = 1, −3   ב. Δ = 4, x = (6 ± 2)/2 → x = 4, 2   ג. Δ = 0, x = −2 (פתרון יחיד)
שאלה 7. א. Δ = 16 + 48 = 64, x = (4 ± 8)/4 → x = 3, −1   ב. Δ = 25 + 24 = 49, x = (5 ± 7)/6 → x = 2, −1/3   ג. Δ = 49 − 24 = 25, x = (−7 ± 5)/4 → x = −1/2, −3
שאלה 8. א. פירוק: (x − 3)(x − 5) = 0 → x = 3, 5   ב. נוסחה: Δ = 4 + 16 = 20, x = 1 ± √5
שאלה 9. א. Δ = 9 > 0 → 2 פתרונות, הפרבולה חותכת פעמיים   ב. Δ = 0 → פתרון יחיד, נוגעת בנקודה אחת   ג. Δ = −5 < 0 → אין פתרון, אינה נוגעת בציר
שאלה 10. Δ = 100 − 100 = 0 → פתרון יחיד x = 10/2 = 5; המשוואה היא (x − 5)² = 0.
שאלה 11. Δ = 36 − 4c = 0 → 4c = 36 → c = 9.
שאלה 12. Δ = 4 − 20 = −16 < 0 → אין פתרון ממשי. הפרבולה y = x² + 2x + 5 כולה מעל ציר ה-x ואינה חותכת אותו.
