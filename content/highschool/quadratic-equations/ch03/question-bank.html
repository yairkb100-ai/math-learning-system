חלק א' — סכום ומכפלה 🎯 (שאלות 1–5)

  1. מצאו סכום ומכפלת השורשים של x² − 9x + 20 = 0 (בלי לפתור).
  2. מצאו סכום ומכפלת השורשים של x² + 4x − 5 = 0.
  3. מצאו סכום ומכפלת השורשים של x² − 2x − 15 = 0.
  4. מצאו סכום ומכפלת השורשים של 2x² − 8x + 6 = 0 (שימו לב ל-a).
  5. מהו סכום השורשים של x² − 11x + 30 = 0?

חלק ב' — מציאת השורשים בעזרת וייטה 🔍 (שאלות 6–10)

  1. מצאו את שורשי x² − 7x + 12 = 0 (איזה זוג נותן סכום 7 ומכפלה 12).
  2. מצאו את שורשי x² − x − 20 = 0.
  3. מצאו את שורשי x² + 8x + 15 = 0.
  4. אחד השורשים של x² − 10x + c = 0 הוא 3. מצאו את השורש השני ואת c.
  5. אחד השורשים של x² + bx + 18 = 0 הוא 6. מצאו את b ואת השורש השני.

חלק ג' — בניית משוואה משורשים 🏗️ (שאלות 11–15)

  1. בנו משוואה ששורשיה 2 ו-7.
  2. בנו משוואה ששורשיה −3 ו-4.
  3. בנו משוואה ששורשיה −2 ו-−5.
  4. בנו משוואה ששורשיה 0 ו-6.
  5. בנו משוואה בעלת פתרון כפול (יחיד) x = 3.

חלק ד' — בעיות מילוליות 📖 (שאלות 16–20)

  1. מכפלת שני מספרים עוקבים היא 72. מצאו את המספרים (החיוביים).
  2. שטח מלבן 54 סמ"ר, אורכו גדול מרוחבו ב-3 ס"מ. מצאו את הממדים.
  3. מספר ומרובעו מסתכמים ב-42. מצאו את המספר החיובי (x² + x = 42).
  4. גובה כדור h = −5t² + 30t. מתי הוא חוזר לקרקע?
  5. אורך גינה ריבועית מוגדל ב-4 מ' וכך מתקבל מלבן ששטחו 45 מ"ר. מהי צלע הריבוע? (x(x + 4) = 45)

חלק ה' — אתגר 🔥 (שאלות 21–25)

  1. עבור x² − 6x + 4 = 0 מצאו את x₁² + x₂² (בעזרת (x₁+x₂)² − 2x₁x₂).
  2. סכום שני מספרים 10 ומכפלתם 21. מצאו אותם (בנו משוואה).
  3. נכון/לא נכון: אם מכפלת השורשים חיובית והסכום שלילי, שני השורשים שליליים. נמקו.
  4. עבור x² + kx + 9 = 0 בעל פתרון יחיד, מצאו את k.
  5. מצאו שני מספרים שהפרשם 2 ומכפלתם 35 (שני פתרונות).
