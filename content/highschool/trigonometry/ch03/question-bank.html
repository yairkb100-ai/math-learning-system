חלק א' — זווית שלישית ומבוא 🔺 (שאלות 1–5)

  1. A = 50°, B = 60°. מצאו C.
  2. A = 25°, B = 100°. מצאו C.
  3. נסחו במילים את משפט הסינוסים.
  4. מה שווה a/sin A לפי המשפט?
  5. מול איזו זווית נמצאת הצלע הארוכה במשולש?

חלק ב' — מציאת צלע (זז"צ) 📏 (שאלות 6–10)

  1. A = 40°, B = 60°, a = 6. מצאו b (sin 40° ≈ 0.643, sin 60° ≈ 0.866).
  2. A = 35°, B = 90°, a = 5. מצאו b (sin 35° ≈ 0.574).
  3. A = 50°, C = 70°, a = 10. מצאו c (sin 50° ≈ 0.766, sin 70° ≈ 0.940).
  4. A = 30°, B = 45°, a = 8. מצאו b (sin 45° ≈ 0.707).
  5. A = 20°, B = 30°, b = 4. מצאו a (sin 20° ≈ 0.342, sin 30° = 0.5).

חלק ג' — מציאת זווית 🔍 (שאלות 11–14)

  1. a = 10, A = 50°, b = 8. מצאו B חדה (sin 50° ≈ 0.766).
  2. a = 12, A = 40°, c = 9. מצאו C חדה (sin 40° ≈ 0.643).
  3. a = 6, A = 30°, b = 9. מצאו את sin B.
  4. כשמוצאים זווית עם sin⁻¹, מדוע יש לזכור את האפשרות הקהה?

חלק ד' — מקרה דו-משמעי ורדיוס ⚖️ (שאלות 15–19)

  1. a = 8, b = 11, A = 35°. מצאו את כל ערכי B (sin 35° ≈ 0.574).
  2. a = 15, b = 10, A = 40°. האם יש מקרה דו-משמעי? נמקו.
  3. a = 7, A = 45°. מצאו 2R (sin 45° ≈ 0.707).
  4. מדוע במקרה צצ"ז ייתכנו שני משולשים?
  5. a = 12, A = 60°. מצאו את רדיוס המעגל החוסם R.

חלק ה' — משולש שלם ואתגר 🔥 (שאלות 20–24)

  1. A = 40°, B = 60°, a = 10. מצאו C, b, c.
  2. A = 30°, a = 6, b = 8. מצאו B חדה, C ו-c.
  3. נכון/לא נכון: אם A + B′ ≥ 180° אין פתרון שני. נמקו.
  4. מדוע כדאי להתחיל מחישוב הזווית השלישית כשנתונות שתי זוויות?
  5. צלע קטנה מול זווית גדולה — האם ייתכן? נמקו.

12. sin C = 9 · 0.643 / 12 ≈ 0.482 → C ≈ 28.8°.
13. sin B = 9 · 0.5 / 6 = 0.75.
14. כי sin B = sin(180° − B) — ייתכנה זווית קהה שגם היא פותרת.
15. sin B = 11 · 0.574 / 8 ≈ 0.789 → B ≈ 52.1° או B′ ≈ 127.9°; 35 + 127.9 = 162.9 < 180 → שתיהן.
16. sin B = 10 · 0.643 / 15 ≈ 0.429 → B ≈ 25.4°; B′ ≈ 154.6° אך 40 + 154.6 > 180 → פתרון יחיד.
17. 2R = 7 / 0.707 ≈ 9.90.
18. כי sin B נתון, ולזווית חדה ולקהה המשלימה יש אותו סינוס.
19. 2R = 12 / sin 60° ≈ 13.86 → R ≈ 6.93.
20. C = 80°; b = 10 · sin 60° / sin 40° ≈ 13.47; c = 10 · sin 80° / sin 40° ≈ 15.32.
21. sin B = 8 · 0.5 / 6 ≈ 0.667 → B ≈ 41.8°; C ≈ 108.2°; c = 6 · sin 108.2° / 0.5 ≈ 11.40.
22. נכון — אם A + B′ ≥ 180° המשולש השני בלתי אפשרי.
23. כי כך מתקבל זוג מלא (צלע והזווית שמולה) לכל צלע נוספת.
24. לא — צלע קטנה נמצאת תמיד מול זווית קטנה.
