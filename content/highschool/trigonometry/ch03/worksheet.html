טריגונומטריה · פרק 3 — משפט הסינוסים והמקרה הדו-משמעי
שם: כיתה: תאריך:
משפט הסינוסים: a/sin A = b/sin B = c/sin C = 2R. מתאים כשיש זוג מלא (צלע והזווית שמולה) ועוד נתון. במקרה צצ"ז (שתי צלעות וזווית מול אחת) — בדקו אפשרות של זווית קהה: B′ = 180° − B.

שאלה 2 — זז"צ — מציאת צלע קל

נתונים A = 30°, B = 70°, a = 5 ס"מ. חשבו את b בעזרת b = a·sin B / sin A. רמז: sin 70° ≈ 0.9397, sin 30° = 0.5.

שאלה 3 — מציאת צלע נוספת בינוני

נתונים a = 12, A = 45°, B = 60°. חשבו את b. רמז: sin 60° ≈ 0.866, sin 45° ≈ 0.7071.

שאלה 4 — מציאת צלע שלישית מלאה בינוני

נתונים A = 40°, B = 75°, a = 8. מצאו את C ואת הצלע c. רמז: sin 40° ≈ 0.643, sin 65° ≈ 0.906.

שאלה 5 — מציאת זווית בינוני

נתונים a = 10, A = 40°, b = 8. מצאו את הזווית B (חדה). רמז: sin B = b·sin A / a, sin 40° ≈ 0.643.

שאלה 6 — המקרה הדו-משמעי מאתגר

נתונים a = 10, b = 14, A = 38°. מצאו את כל הערכים האפשריים של B. חשבו sin B, מצאו זווית חדה, ובדקו אם B′ = 180° − B אפשרית. רמז: sin 38° ≈ 0.6157.

שאלה 7 — רדיוס המעגל החוסם מאתגר

נתונים a = 9, A = 50°. חשבו את רדיוס המעגל החוסם R (מ-2R = a/sin A). רמז: sin 50° ≈ 0.766.

שאלה 8 — מתי סינוסים? קל

לכל מקרה כתבו אם משפט הסינוסים מתאים (כן/לא):

  1. שתי זוויות וצלע ⟵
  2. שלוש צלעות ⟵
  3. שתי צלעות והזווית מול אחת מהן ⟵

שאלה 9 — נכון או לא נכון — נמקו בינוני

קבעו נכון/לא נכון ונמקו:

  1. מול הזווית הגדולה נמצאת הצלע הארוכה.
  2. משפט הסינוסים תקף רק במשולש ישר-זווית.
  3. במקרה צצ"ז ייתכנו שני משולשים.

שאלה 10 — אתגר — משולש שלם מאתגר

נתונים A = 35°, a = 7, b = 10. מצאו את B (חדה), את C ואת c. רמז: sin 35° ≈ 0.5736.
