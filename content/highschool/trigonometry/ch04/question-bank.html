25 שאלות · טריגונומטריה · פרק 4 — משפט הקוסינוסים כהכללת פיתגורס · דף פתרונות בעמוד האחרון
שם: כיתה: תאריך:

חלק א' — הנוסחה ופיתגורס 📐 (שאלות 1–5)

  1. כתבו את משפט הקוסינוסים עבור הצלע c.
  2. מדוע משפט הקוסינוסים הוא הכללה של פיתגורס?
  3. a = 6, b = 8, C = 90°. מצאו c.
  4. כתבו נוסחה ל-cos C כאשר נתונות שלוש הצלעות.
  5. אם c² > a² + b², איזו זווית היא C?

חלק ב' — צז"צ 📏 (שאלות 6–10)

  1. a = 5, b = 7, C = 60°. מצאו c (cos 60° = 0.5).
  2. a = 10, b = 6, C = 45°. מצאו c (cos 45° ≈ 0.707).
  3. a = 8, b = 8, C = 60°. מצאו c.
  4. a = 4, b = 9, C = 120°. מצאו c (cos 120° = −0.5).
  5. a = 12, b = 5, C = 90°. מצאו c.

חלק ג' — שלוש צלעות → זווית 🔍 (שאלות 11–15)

  1. צלעות 4, 5, 6. מצאו את הזווית מול 6.
  2. צלעות 7, 8, 9. מצאו את הזווית מול 9.
  3. צלעות 6, 8, 10. מצאו את הזווית מול 10.
  4. צלעות 5, 12, 13. מצאו את הזווית מול 13.
  5. צלעות 3, 5, 7. מצאו את הזווית מול 7 (cos⁻¹(−0.5) = 120°).

חלק ד' — סיווג זוויות ⚖️ (שאלות 16–20)

  1. a = 6, b = 8, c = 9 — הזווית מול c חדה/ישרה/קהה?
  2. a = 5, b = 5, c = 8 — הזווית מול c חדה/ישרה/קהה?
  3. a = 8, b = 15, c = 17 — הזווית מול c?
  4. מדוע במשפט הקוסינוסים אין מקרה דו-משמעי?
  5. כשנתונות שלוש צלעות, איזו זווית כדאי לחשב קודם ולמה?

חלק ה' — יישום ואתגר 🔥 (שאלות 21–25)

  1. מקבילית: צלעות 5 ו-8, זווית 60°. מצאו את האלכסון מול 60°.
  2. שני כבישים יוצאים מצומת בזווית 70°; מכוניות נסעו 40 ו-55 ק"מ. מה המרחק ביניהן? (cos 70° ≈ 0.342)
  3. צלעות משולש 9, 10, 17. מצאו את הזווית הגדולה.
  4. נכון/לא נכון: אם כל הצלעות שוות, כל הזוויות 60°. נמקו.
  5. מדוע כשנתונות שתי צלעות והזווית ביניהן אי אפשר להשתמש ישירות במשפט הסינוסים?

למורה / להורה / לבדיקה עצמית
1. c² = a² + b² − 2ab·cos C.
2. כי כאשר C = 90°, cos C = 0 והנוסחה הופכת ל-c² = a² + b².
3. c² = 36 + 64 − 0 = 100 → c = 10.
4. cos C = (a² + b² − c²) / 2ab.
5. קהה (cos C < 0).
6. c² = 25 + 49 − 70·0.5 = 39 → c ≈ 6.24.
7. c² = 100 + 36 − 120·0.707 ≈ 51.2 → c ≈ 7.15.
8. c² = 64 + 64 − 128·0.5 = 64 → c = 8 (שווה-צלעות).
9. c² = 16 + 81 − 72·(−0.5) = 133 → c ≈ 11.53.
10. c² = 144 + 25 = 169 → c = 13.
11. cos θ = (16 + 25 − 36)/40 = 0.125 → θ ≈ 82.8°.
12. cos θ = (49 + 64 − 81)/112 ≈ 0.286 → θ ≈ 73.4°.
13. cos θ = (36 + 64 − 100)/96 = 0 → θ = 90°.
14. cos θ = (25 + 144 − 169)/120 = 0 → θ = 90°.
15. cos θ = (9 + 25 − 49)/30 = −0.5 → θ = 120°.
16. cos θ = (36 + 64 − 81)/96 ≈ 0.198 → חדה (θ ≈ 78.6°).
17. cos θ = (25 + 25 − 64)/50 = −0.28 → קהה.
18. 64 + 225 = 289 = 17² → ישרה (90°).
19. כי cos של זווית קהה שלילי, ולכן cos⁻¹ מבחין ומחזיר זווית יחידה.
20. את הזווית הגדולה (מול הצלע הארוכה); אם היא יצאה, השאר בוודאי חדות.
21. d² = 25 + 64 − 80·0.5 = 49 → d = 7.
22. d² = 1600 + 3025 − 4400·0.342 ≈ 3120 → d ≈ 55.9 ק"מ.
23. cos θ = (81 + 100 − 289)/180 = −0.6 → θ ≈ 126.9°.
24. נכון — משולש שווה-צלעות, cos θ = (2a² − a²)/2a² = 0.5 → 60°.
25. כי משפט הסינוסים דורש זוג מלא (צלע והזווית שמולה), ואין כזה בנתוני צז"צ.
