טריגונומטריה · פרק 4 — משפט הקוסינוסים כהכללת פיתגורס
שם: כיתה: תאריך:
משפט הקוסינוסים: c² = a² + b² − 2ab·cos C (הזווית מול הצלע שמשמאל). מציאת זווית משלוש צלעות: cos C = (a² + b² − c²) / 2ab. כש-C = 90° מתקבל פיתגורס. אין דו-משמעות — cos⁻¹ מחזיר זווית יחידה בין 0° ל-180°.

שאלה 1 — בדיקת פיתגורס קל

נתונים a = 3, b = 5, C = 90°. חשבו את c בעזרת משפט הקוסינוסים וודאו שהתקבל פיתגורס. רמז: cos 90° = 0.

שאלה 2 — צז"צ — מציאת צלע בינוני

נתונים a = 8, b = 11, C = 40°. חשבו את c. רמז: cos 40° ≈ 0.766.

שאלה 3 — צז"צ נוסף בינוני

נתונים a = 7, b = 10, C = 60°. חשבו את c. רמז: cos 60° = 0.5.

שאלה 4 — שלוש צלעות — מציאת זווית בינוני

צלעות משולש 7, 9, 12. מצאו את הזווית שמול הצלע 12 (מ-cos θ = (a² + b² − c²)/2ab).

שאלה 5 — זווית חדה או קהה? בינוני

לכל משולש קבעו (בלי מחשבון) אם הזווית מול הצלע הארוכה חדה, ישרה או קהה, לפי השוואת c² ל-a² + b²:

  1. a = 3, b = 4, c = 5
  2. a = 3, b = 4, c = 6
  3. a = 5, b = 5, c = 6

שאלה 6 — מציאת הזווית הגדולה מאתגר

צלעות משולש 5, 8, 10. מצאו את הזווית הגדולה ביותר (מול הצלע 10).

שאלה 7 — אלכסון מקבילית מאתגר

במקבילית צלעות 6 ו-9 ס"מ וזווית 70° ביניהן. חשבו את שני האלכסונים (האחד מול 70°, השני מול 110°). רמז: cos 70° ≈ 0.342, cos 110° ≈ −0.342.

שאלה 8 — בחירת המשפט קל

לכל נתון סמנו איזה משפט נוח (סינוסים / קוסינוסים):

  1. שתי צלעות והזווית ביניהן ⟵
  2. שלוש צלעות ⟵
  3. זווית, צלע מולה, ועוד זווית ⟵

שאלה 9 — נכון או לא נכון — נמקו בינוני

קבעו נכון/לא נכון ונמקו:

  1. משפט הקוסינוסים הוא הכללה של פיתגורס.
  2. אם cos C < 0 הזווית C קהה.
  3. במציאת זווית משלוש צלעות יש לבדוק שתי אפשרויות.

שאלה 10 — אתגר — משולש שלם מ-3 צלעות מאתגר

צלעות 6, 8, 9. מצאו את הזווית מול הצלע 9, ואז השתמשו במשפט הסינוסים למציאת עוד זווית.
