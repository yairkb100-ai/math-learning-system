דף עבודה — פרק 5: שטח משולש ובעיות משולבות

טריגונומטריה · פרק 5 — נוסחת השטח הטריגונומטרית ובעיות משולבות
שם: כיתה: תאריך:
שטח משולש: S = ½·a·b·sin C — מחצית מכפלת שתי צלעות בסינוס הזווית שביניהן. הפוך: sin C = 2S/(ab) (זהירות — ייתכנו זווית חדה וקהה). נוסחת הרון: S = √(p(p−a)(p−b)(p−c)), p = חצי ההיקף.

שאלה 1 — שטח בסיסי קל

חשבו את שטח המשולש (צלעות והזווית שביניהן):

  1. a = 8, b = 5, C = 60° (sin 60° ≈ 0.866)
  2. a = 4, b = 10, C = 30° (sin 30° = 0.5)

שאלה 2 — שטח נוסף קל

חשבו את השטח:

  1. a = 6, b = 6, C = 60°
  2. a = 12, b = 7, C = 90°
  3. a = 9, b = 4, C = 45° (sin 45° ≈ 0.707)

שאלה 3 — משולש שווה-צלעות בינוני

חשבו את שטח משולש שווה-צלעות שצלעו 6 ס"מ בעזרת הנוסחה הטריגונומטרית (S = ½·6·6·sin 60°).

שאלה 4 — מציאת זווית מתוך שטח בינוני

שטח משולש 20 סמ"ר, צלעותיו 9 ו-7 ס"מ. מצאו את כל הערכים האפשריים של הזווית ביניהן (sin C = 2S/(ab)).

שאלה 5 — מציאת צלע מתוך שטח בינוני

שטח משולש 30 סמ"ר, אחת מצלעותיו 10 ס"מ והזווית הצמודה C = 40°. מצאו את הצלע השנייה (a = 2S/(b·sin C), sin 40° ≈ 0.643).

שאלה 6 — בעיה משולבת — סינוסים ואז שטח מאתגר

נתונים a = 8, A = 40°, B = 75°. מצאו את b (סינוסים), את C, ואז את השטח S = ½·a·b·sin C. רמז: sin 40° ≈ 0.643, sin 75° ≈ 0.966, sin 65° ≈ 0.906.

שאלה 7 — בעיה משולבת — קוסינוסים ואז שטח מאתגר

נתונים a = 9, b = 7 והזווית C חדה כך שהשטח 20 סמ"ר. מצאו את C ואז את הצלע c (משפט הקוסינוסים). רמז: השתמשו ב-sin C = 40/63.

שאלה 8 — נוסחת הרון בינוני

חשבו בעזרת נוסחת הרון את שטח המשולש שצלעותיו 5, 6, 7 (p = 9).

שאלה 9 — נכון או לא נכון — נמקו בינוני

קבעו נכון/לא נכון ונמקו:

  1. בנוסחה S = ½ab·sin C הזווית C היא בין הצלעות a ו-b.
  2. אם sin C = 0.5 אז C יכולה להיות 30° או 150°.
  3. כשהזווית בין שתי צלעות קבועות גדלה מ-0° ל-90°, השטח גדל.

שאלה 10 — אתגר — שטח מרבי מאתגר

שתי צלעות משולש קבועות: a = 6, b = 8. עבור איזו זווית C השטח מרבי, ומהו השטח? (רמז: sin C מרבי כאשר C = 90°).
