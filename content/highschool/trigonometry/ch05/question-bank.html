חלק א' — שטח בסיסי 📐 (שאלות 1–5)

  1. a = 10, b = 4, C = 30°. מצאו את השטח.
  2. a = 6, b = 9, C = 90°. מצאו את השטח.
  3. a = 8, b = 8, C = 60°. מצאו את השטח (sin 60° ≈ 0.866).
  4. a = 5, b = 12, C = 45°. מצאו את השטח (sin 45° ≈ 0.707).
  5. כתבו את נוסחת השטח הטריגונומטרית.

חלק ב' — שטח מיוחד 🔺 (שאלות 6–9)

  1. שטח משולש שווה-צלעות שצלעו 8 (S = ½·8·8·sin 60°).
  2. a = 7, b = 7, C = 90°. מצאו את השטח.
  3. a = 10, b = 10, C = 120°. מצאו את השטח (sin 120° ≈ 0.866).
  4. מדוע כשהזווית 90° השטח מקסימלי לצלעות נתונות?

חלק ג' — הפוך: זווית או צלע 🔍 (שאלות 10–13)

  1. שטח 24, צלעות 8 ו-8. מצאו את הזווית (sin C = 2S/ab).
  2. שטח 15, צלעות 6 ו-10. מצאו את הזווית (שתי אפשרויות).
  3. שטח 36, צלע 12, זווית צמודה 30°. מצאו את הצלע השנייה.
  4. שטח 50, צלעות 20 ו-10. מצאו את הזווית (sin C = 0.5).

חלק ד' — נוסחת הרון ובחירה 📊 (שאלות 14–18)

  1. נוסחת הרון לצלעות 6, 8, 10 (p = 12).
  2. נוסחת הרון לצלעות 3, 4, 5 (p = 6).
  3. מהי נוסחת השטח עם רדיוס המעגל החוסם R?
  4. נתונות שלוש צלעות — איזו נוסחת שטח ישירה?
  5. כתבו את נוסחת הרון והגדירו את p.

חלק ה' — בעיות משולבות ואתגר 🔥 (שאלות 19–23)

  1. a = 6, A = 30°, B = 90°. מצאו b, C והשטח.
  2. a = 10, b = 6, C = 60°. מצאו את הצלע c ואת השטח.
  3. שתי צלעות a = 5, b = 9 קבועות. עבור איזו C השטח מרבי, ומהו?
  4. משולש שצלעותיו 5, 6, 7 — מצאו את שטחו (הרון) ואת הזווית מול 7.
  5. נכון/לא נכון: לשני משולשים עם אותן שתי צלעות ואותו שטח, בהכרח אותה זווית. נמקו.
